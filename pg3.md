# API

## Rest

Uma API Rest (*Representational State Transfer*) é uma arquitetura de comunicação entre sistemas. Ela utiliza o protocolo HTTP para criar, ler, atualizar e excluir (CRUD). Essas representações são entregues de várias maneiras, sendo as mais comuns **JSON/XML**.



---

## Métodos do HTTP

| Método   | Ação       | CRUD            |
|----------|------------|-----------------|
| POST     | Criar      | C - *create*    |
| GET      | Consultar  | R - *read*      |
| PUT      | Atualizar  | U - *update*    |
| DELETE   | Deletar    | D - *delete*    |
