# Função Callback

Permite que funções sejam passadas como argumentos para outras funções e executadas posteriormente.

---

## Exemplos:

### Método `.map()` — Retorna um novo array com elementos modificados:


```javascript
let newArray = array.map((elemento, indice, array) => {
  return // código
});
```

---

### Método `.filter()` — Seleciona os elementos que atendem uma condição específica:

```javascript
let arrayFiltrada = array.filter((elemento, indice) => {
  return // condição ? true : false
});
```

---

### Método `.reduce()` — Utilizado para somar todos os elementos de uma array:

```javascript
let somaArray = array.reduce((acumulador, elemento) => {
  return acumulador + elemento;
}, 0);
```

> 💡 **O `0` no final** é o valor inicial do acumulador — sempre defina para evitar comportamentos inesperados.

> 💡 **Resumo dos 3 métodos:**
> | Método | O que faz | Retorna |
> |---|---|---|
> | `.map()` | Transforma cada elemento | Novo array do mesmo tamanho |
> | `.filter()` | Filtra por condição | Novo array menor ou igual |
> | `.reduce()` | Acumula em um único valor | Um único valor (número, string, objeto...) |
