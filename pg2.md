# Endereço Sabe seu Domínio

Um endereço de internet é uma sequência de números e letras que identifica de forma exclusiva um host (computador, dispositivo, servidor).

- Para verificar o endereço IPv4/IPv6: no terminal digite o `nslookup exemplo.com`

---

## Domínio

É um nome identificador único na web, registrado por uma pessoa ou empresa, que aponta para um endereço IP específico na internet. Exemplos de domínios de internet incluem "google.com" e "wikipedia.org".

- **Subdomínio:** É uma seção separada de um domínio de internet que pode ser usada para organizar ou agrupar recursos relacionados. É criado adicionando um prefixo ao nome do domínio principal.

> Exemplo: mail.google.com

---

> "URL (*Uniform Resource Locator*): são endereços que identificam a localização de qualquer recurso na internet, como páginas web = (www.site.com.br)"

## O cliente pede e o Servidor Responde

O conceito de Cliente-Servidor é um modelo de arquitetura de computadores onde o servidor oferece recursos, serviços e informações para um ou mais clientes. O cliente envia uma solicitação e o servidor responde.

> "Cookie é um pequeno arquivo texto armazenado no navegador de um usuário quando ele visita um site"
