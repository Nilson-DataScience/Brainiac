# Separando o Conteúdo e Informação

```html
<!DOCTYPE html>
<html lang="idioma"> <!-- en, pt-br -->

  <head>

    <!-- Informação para o Navegador -->

  </head>

  <body>

    <!-- Conteúdo que o usuário vai visualizar -->

  </body>

</html>
```

---

## Atividades

**1.** Para criar o site do Livro, deve ser criados títulos, parágrafos, imagens, isso para o glossário e um rodapé para escrever o nome do criador. Onde essas tags devem ser criadas — no `<head>` ou no `<body>`?

> **R:** `<body>`, pois é nessa tag que armazena o conteúdo.

**2.** Qual tag de parágrafo?
> `<p>...</p>`

**3.** Qual tag para título?
> `<h1>...</h1>` *(até `<h6>`)*


**4.** Qual tag para imagem?
> `<img src="" alt="">`
