# Alterando Arrays

---

## Dividindo um Array com `.slice(indiceInicial, indiceFinal)`

```javascript
const soloLeveling = ["Sung Jinwoo", "Cha Hae-In", "Beru"];

let timeA = soloLeveling.slice(0, 1); // ["Sung Jinwoo"]
let timeB = soloLeveling.slice(1, 3); // ["Cha Hae-In", "Beru"]
```

> 💡 **Lembre-se:** `.slice()` **não modifica** o array original — retorna uma cópia.
> O índice final é **exclusivo**: `slice(1, 3)` pega os índices 1 e 2, não o 3.

---

## Adicionar, Remover ou Substituir com `.splice()`

```javascript
let index = 0;
let inicio = index;
let quantidadeRemovida = 0;
let elemento = "Bae";

timeA.splice(inicio, quantidadeRemovida, elemento);
```

> 💡 **Sintaxe:** `array.splice(início, quantidadeRemovida, ...elementosNovos)`
> - Se `quantidadeRemovida = 0`, apenas **adiciona** sem remover nada.
> - Se `quantidadeRemovida > 0`, **remove** e opcionalmente substitui.

---

## Concatenando Arrays

```javascript
let timeOp = timeA.concat(timeB);
```

> 💡 **Alternativa moderna com spread:**
> ```javascript
> let timeOp = [...timeA, ...timeB];
> ```

---

## Contando Elementos e Caracteres

```javascript
timeOp.length
```

**Qual array é maior — `timeOp` ou `soloLeveling`?**
> **R:** `timeOp`

> ⚠️ **Nota:** `timeOp` = `timeA` + `timeB` = todos os elementos de `soloLeveling` + o elemento `"Bae"` adicionado via `.splice()`, então `timeOp.length > soloLeveling.length`.
