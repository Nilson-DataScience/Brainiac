# Gênero Textual: Receita

## O Primeiro Passo

**→ Acessar o GitHub** e criar um repositório `"Quatro-Vidas-de-um-Cachorro"`. Clicar em **"."** no teclado.

**→ Automaticamente o VSCode vai abrir**, no lado esquerdo no explorador (`CTRL + Shift + E`).

**→ No explorador no lado direito** ou no nome `quatro-vidas-de-um-cachorro` tem um menu de criação:

```
 [ 📄 ]        [ 📁 ]       [ ↺ ]
Novo          Nova          Atualizar
Arquivo       Pasta         Explorador
```

**→ Clique em Novo Arquivo** e renomeie como `index.html`.  
Depois, dentro do arquivo `index.html`, digite `html:5` e **Enter**.
