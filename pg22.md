# Laço de Repetição

---

## `for` — Clássico

É usado quando o número de iterações é conhecido. Depois de informada a quantidade de iterações, ela não pode ser modificada.

```javascript
for (let index = 0; index < quantidadeIteracoes; index++) {
  // código
}
```

> 💡 **As 3 partes do `for`:**
> - `let index = 0` → **início** — onde começa
> - `index < quantidadeIteracoes` → **condição** — até quando repete
> - `index++` → **incremento** — o que muda a cada volta

---

## `for...of`

Itera sobre os **valores** de um array diretamente, sem precisar do índice.

```javascript
for (let elemento of array) {
  // código
}
```

---

## `.forEach()`

Método de array que itera sobre cada elemento com uma função callback.

```javascript
array.forEach((elemento, index, array) => {
  // código
});
```

```javascript
while(condicao){
    // Para parar precisa que à condição seja falsa
    // ou  if com break 
}
```

> 💡 **Diferença entre `for...of` e `.forEach()`:**
> - `for...of` → mais simples, funciona com qualquer iterável (arrays, strings, maps...)
> - `.forEach()` → exclusivo de arrays, dá acesso ao índice e ao array original como parâmetros extras
> - Nenhum dos dois retorna valor — use `.map()` se precisar de um novo array transformado

> 💡 **Complemento — outros laços importantes:**
> - **`while`** → repete enquanto a condição for verdadeira (número de iterações desconhecido)
> - **`do...while`** → igual ao `while`, mas executa pelo menos uma vez antes de verificar a condição
> - **`for...in`** → itera sobre as **chaves** de um objeto
