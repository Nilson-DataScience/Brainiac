```javascript
const matriz = [[1, 2], [3, 4]];
```

```
| 1 | 2 |
| 3 | 4 |
```

# Laço de Repetição

---

## `for` — Clássico

É usado quando o número de iterações é conhecido. Depois de informada a quantidade de iterações, ela não pode ser modificada.

```javascript
for (let index = 0; index < quantidadeIteracoes; index++) {
  // código
}
```



---

## `for...of`


```javascript
for (let elemento of array) {
  // código
}
```

---

## `.forEach()`

```javascript
array.forEach((elemento, index, array) => {
  // código
});
```
