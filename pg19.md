# Funções em JavaScript

---

## Função Padrão

```javascript
function nome(parametro) {
  // Código
  return resultado;
}

nome(argumento);
```

---

## Arrow Function

```javascript
const flecha = (parametro) => {
  return resultado;
}
```



---

## Exercício: Criar 2 funções — padrão e arrow

**Função padrão:**
```javascript
function soma(a, b) {
  return a + b;
}
```

**Arrow function:**
```javascript
const soma = (a, b) => {
  return a + b;
}
```
