# Funções em JavaScript

---

## Função Normal

```javascript
function circulo(parametro) {
  // Código
  return resultado;
}

circulo(argumento);
```

---

## Arrow Function

```javascript
const flecha = (parametro) => {
  return resultado;
}
```

> 💡 **Complemento:** Quando a arrow function tem apenas **uma expressão**, pode ser simplificada em uma linha (retorno implícito):
> ```javascript
> const flecha = (parametro) => resultado;
> ```

---

## Exercício: Criar 2 funções — normal e arrow

**Função normal:**
```javascript
function soma(a, b) {
  return a + b;
}
```

**Arrow function:**
```javascript
const soma = (a, b) => {
  return a + b;
}
```

## b) Implementando Ternários nas Funções Arrow

**Regra de Negócio:** Robin Hood — decide roubar dos ricos para dar aos pobres.

```javascript
const robinHood = (fortuna) => {
  return fortuna > 40000 ? "Rico" : "Pobre";
}
```

> 💡 **Complemento — retorno implícito com ternário:**
> ```javascript
> const robinHood = (fortuna) => fortuna > 40000 ? "Rico" : "Pobre";
> ```
