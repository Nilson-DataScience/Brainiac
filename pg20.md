# O que são Arrays?

São coleções de elementos, cada um com um índice numérico.

**Exemplo:**
```javascript
const mapaSala = [
  ["Naruto", "Sasuke", "Sakura"],  // índice 0
  ["Hashirama"]                     // índice 1
];
```

---

## Métodos Array

### Acessar por Índice:

```javascript
mapaSala[0] // ["Naruto", "Sasuke", "Sakura"]
```

---

### Adicionando Itens:

```javascript
mapaSala[1].push("Naruto")      // adiciona no final
mapaSala[1].unshift("Elisui")   // adiciona no início

// Resultado: ["Elisui", "Hashirama", "Naruto"]
```

---

### Deletando Elementos:

```javascript
mapaSala[0].shift()  // remove o primeiro elemento
mapaSala[0].pop()    // remove o último elemento
```
