# O que são Objetos?

Em JavaScript, objetos são coleções de propriedades, que são pares de **chave-valor**.

```javascript
const obj = { key: value }
```

---

## Acessando Dados

```javascript
obj.key        // notação de ponto
obj["key"]     // notação de colchetes
```

> 💡 **Quando usar cada um:**
> - `.key` → uso geral, mais legível
> - `["key"]` → quando a chave é dinâmica (variável) ou tem caracteres especiais

---

## Adicionando e Alterando

```javascript
obj.key = "value"   // adiciona se não existir, altera se já existir
```

---

## Deletar Propriedade

```javascript
delete obj.key
```

---

## Manipulando Objetos

### Objeto + Array:

```javascript
let objeto = { key: [value, value2] }
```

### Objeto + Objeto:

```javascript
const objeto = {
  key1: value,
  key2: value
}
```

> 💡 **Complemento — métodos úteis de Objeto:**
> ```javascript
> Object.keys(obj)    // retorna array com as chaves
> Object.values(obj)  // retorna array com os valores
> Object.entries(obj) // retorna array de pares [chave, valor]
> ```

> 💡 **Complemento — spread em objetos:**
> ```javascript
> const novoObj = { ...obj, novaKey: "novoValor" }
> ```
> Copia todas as propriedades de `obj` e adiciona/sobrescreve com as novas.
