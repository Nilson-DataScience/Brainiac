# Depuração das Requisições HTTP

É o processo de verificação de erros e solução de problemas relacionados ao protocolo HTTP durante a comunicação entre um servidor e um cliente.

---

## Código de Respostas

Os códigos de resposta HTTP são números de 3 dígitos retornados pelo servidor em resposta.

### Código de Status:

> - **2xx** → Sucesso (ex: `201 Created`)
> - **3xx** → Redirecionamento (ex: `301 Moved Permanently`)
> - **4xx** → Erro do cliente (ex: `400 Bad Request`, `401 Unauthorized`, `403 Forbidden`)
> - **5xx** → Erro do servidor (ex: `500 Internal Server Error`)
