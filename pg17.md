# JavaScript — NodeJS

**Node.js** é uma plataforma de execução para JavaScript no servidor, baseada no motor **V8** do Chrome.

---

## Ambiente de Execução

Sendo executado no lado do servidor, permite a construção de aplicações Web, APIs e processamento de dados.

**Módulos:** Node.js usa um sistema de módulos para organizar e compartilhar código.

**NPM:** Facilita a instalação e gerenciamento de bibliotecas e dependências.

> 💡 **Complemento — NPM:** *Node Package Manager* é o maior repositório de pacotes do mundo. Comandos básicos:
> ```bash
> npm init        # inicia um projeto Node
> npm install     # instala as dependências do projeto
> npm install pacote  # instala um pacote específico
> ```

**Vatagem (Event Loop):** O *Event Loop* é o coração do Node.js, permitindo operações assíncronas e não bloqueantes. Node.js pode lidar com múltiplas operações.
