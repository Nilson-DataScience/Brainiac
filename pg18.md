# Operadores — JavaScript

---

## Operadores de Comparação

| Operador | Descrição |
|----------|-----------|
| `>`      | Maior que |
| `<`      | Menor que |
| `>=`    | Maior igual|
| `<=`    | Menor igual|
| `===`    |Igual|
| `!==`     | Desigualdade com coerção de tipo |
| `==`     | Igualdade com coerção de tipo |



---

## Operador Ternário

```
condição ? expressão1 : expressão2
               ↑              ↑
          Verdadeiro        Falso
```

> 💡 **Exemplo:**
> ```javascript
> let idade = 18;
> let status = idade >= 18 ? "maior de idade" : "menor de idade";
> ```
