# Operadores — JavaScript

---

## Operadores de Comparação

| Operador | Descrição |
|----------|-----------|
| `>`      | Maior que |
| `<`      | Menor que |
| `!=`     | Desigualdade |
| `!==`    | Desigualdade estrita |
| `===`    | Igualdade estrita |

> ⚠️ **Nota:** Na anotação consta `==` como "desigualdade" — o correto é que `==` é **igualdade com coerção de tipo**. O operador de desigualdade com coerção é `!=`. Abaixo a tabela correta e completa:

| Operador | Descrição |
|----------|-----------|
| `==`     | Igualdade com coerção de tipo |
| `!=`     | Desigualdade com coerção de tipo |
| `===`    | Igualdade estrita (sem coerção) |
| `!==`    | Desigualdade estrita (sem coerção) |

> 💡 **Dica:** Prefira sempre `===` e `!==` no lugar de `==` e `!=`, pois evitam bugs causados pela coerção implícita de tipos.

## Operador Ternário

```
condição ? expressão1 : expressão2
               ↑              ↑
          Verdadeiro        Falso
```

> 💡 **Exemplo:**
> ```javascript
> let idade = 18;
> let status = idade >= 18 ? "maior de idade" : "menor de idade";
> ```

## Template Literals

```javascript
`Olá, ${expressão}!`;
```

> 💡 **Complemento:** Template literals usam **crases** (`` ` ``) e permitem interpolação de variáveis e expressões com `${}`, além de suportar **strings multilinha** sem precisar de `\n`.
