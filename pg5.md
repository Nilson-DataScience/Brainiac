# Criação de Sites

**HTML** é uma linguagem de marcação utilizada para criar páginas da web.

**Como ocorre essa marcação?**
Simples, ocorre por meio das *tags*.

---

## # Semântica HTML

Refere-se ao uso de elementos que possuem significado claro e descritivo, tanto para navegadores e máquinas de busca.

> 💡 **Complemento:** A semântica HTML também é fundamental para **acessibilidade**, ajudando leitores de tela a interpretar corretamente o conteúdo da página.

### Elementos semânticos:

- **`<header>`** — Representa o cabeçalho de um documento ou seção.
- **`<main>`** — Define o conteúdo principal de uma página.
- **`<footer>`** — Representa o rodapé de um documento ou seção.

### Estrutura visual:

```
┌─────────────────┐
│     <header>    │
├─────────────────┤
│     <main>      │
├─────────────────┤
│     <footer>    │
└─────────────────┘
```
