# Tipo Primitivo — JavaScript

## Tipos Primitivos

**Number:** Trata todos os números no sistema de ponto flutuante com 64 bits, segundo o padrão **IEEE 754**.

**String:** São usadas para representar texto e podem ser criadas usando: aspas simples, duplas ou *template literals* (crases).

**Boolean:** Representando `true` ou `false`, são cruciais para lógica condicional e tomada de decisão.

**Undefined:** É um tipo atribuído automaticamente a variáveis não inicializadas.

**Null:** É usado para indicar intencionalmente a ausência de um objeto.

**Symbol:** Cria identificadores únicos, úteis para propriedades de objetos.

**BigInt:** Permite trabalhar com números inteiros maiores do que 2⁵³ - 1.
